# Game execution — Phase 5: Guilds & milestone bosses

Parent: [001-buildfirst](../../001-buildfirst/plan.md) phase 5 · companion: `worldd/plans/002-execution` phase 4.

## Goal

Group play: guilds, async milestone-Warden quorums, the frontier advancing for everyone.

## Deliverables

- `ascent_guild` tool (skill-gated until unlocked at the Guildhall): found (fee), join/leave, roster, guild board.
- Boss commit flow: milestone Warden approach scene → commit card (quorum dots ■■□, committed names, server countdown text) → `boss commit` (5⚡ escrow) → resolution event card on next session (win: payouts + names on Stone; lapse: refund).
- Frontier: tower gate floor list shows world frontier; a milestone clear opens the next floor for every tenant.
- Quorums configurable (small-launch default: Gnarl 2).
- Guild watchtower v1 deferred if time-boxed out — note in summary if so (001 open decision 4).

## Exit gate

Async Gnarl clear by 3 players across 2 tenants in real chat: commits on different turns, resolution fires once, payout cards render, a 4th player sees floor 11 open and names on the Stone.
